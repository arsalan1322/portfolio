Machine Unlearning

Machine learning project, December 2023

This project explores a novel method for "unlearning" previously learned datapoints from a model. Unlearning is the process of having a deployed machine learning model forget about some of its training data points. This can be achieved through exact unlearning methods such as Sharded, Isolated, Sliced, and Aggregated (SISA), which involve formally removing the data point’s impact on the model by retraining the model from scratch; or approximate unlearning, where the model parameters are approximated to save on computational costs.

We implement and test the SISA approach on multiple datasets (CIFAR-10 and MNIST) and multiple neural network architectures. Our work seeks to extend SISA by exploring combinations of SISA with ensemble methods, and to empirically test the effects of hyperparameters on relatively simple computer vision classification tasks.

  • Tools used: Python, TensorFlow, Pytorch.

The full report can be found here where we go further into detail, with a thorough literature review, methodology, results, and discussion.
